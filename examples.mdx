## Complete Payment Flow

Here's a complete example of processing a payment from start to finish.

### 1. Create a Product

```bash
curl -X POST "https://api.klivopay.com.br/api/public/v1/products?api_token=YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Premium Course",
    "product_type": "digital",
    "delivery_type": 1,
    "id_category": 1,
    "amount": 49700
  }'
```

**Response:**
```json
{
  "hash": "abc123",
  "title": "Premium Course",
  "amount": 49700
}
```

### 2. Create an Offer

```bash
curl -X POST "https://api.klivopay.com.br/api/public/v1/products/abc123/offers?api_token=YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Launch Offer",
    "amount": 29700
  }'
```

**Response:**
```json
{
  "hash": "offer123",
  "title": "Launch Offer",
  "amount": 29700
}
```

## Installments Calculation

Calculate installment options before processing payment:

```bash

```

  -H "Content-Type: application/json" \

## SDK Examples

### JavaScript/Node.js

```javascript
const axios = require('axios');

const KLIVOPAY_API_URL = 'https://api.klivopay.com.br/api';
const API_TOKEN = process.env.KLIVOPAY_API_TOKEN;

async function createTransaction(data) {
  const response = await axios.post(
    `${KLIVOPAY_API_URL}/public/v1/transactions`,
    data,
    {
      params: { api_token: API_TOKEN },
      headers: { 'Content-Type': 'application/json' }
    }
  );
  
  return response.data;
}

// Usage
const transaction = await createTransaction({
  amount: 29700,
  offer_hash: 'offer123',
  payment_method: 'credit_card',
  // ... other fields
});
```

### Python

```python
import requests
import os

KLIVOPAY_API_URL = 'https://api.klivopay.com.br/api'
API_TOKEN = os.getenv('KLIVOPAY_API_TOKEN')

def create_transaction(data):
    response = requests.post(
        f'{KLIVOPAY_API_URL}/public/v1/transactions',
        json=data,
        params={'api_token': API_TOKEN}
    )
    return response.json()

# Usage
transaction = create_transaction({
    'amount': 29700,
    'offer_hash': 'offer123',
    'payment_method': 'credit_card',
    # ... other fields
})
```

### PHP

```php
<?php

define('KLIVOPAY_API_URL', 'https://api.klivopay.com.br/api');
define('API_TOKEN', getenv('KLIVOPAY_API_TOKEN'));

function createTransaction($data) {
    $ch = curl_init();
    
    curl_setopt($ch, CURLOPT_URL, KLIVOPAY_API_URL . '/public/v1/transactions?api_token=' . API_TOKEN);
    curl_setopt($ch, CURLOPT_POST, 1);
    curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
    curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/json']);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    
    $response = curl_exec($ch);
    curl_close($ch);
    
    return json_decode($response, true);
}

// Usage
$transaction = createTransaction([
    'amount' => 29700,
    'offer_hash' => 'offer123',
    'payment_method' => 'credit_card',
    // ... other fields
]);
?>
```

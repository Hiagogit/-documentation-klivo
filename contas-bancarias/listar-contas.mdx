## Visão Geral

Este endpoint retorna uma lista de todas as contas bancárias e chaves PIX cadastradas na sua conta KlivoPay.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/bank-accounts
```

## Parâmetros

<ParamField query="api_token" type="string" required>
  Seu token de autenticação da API KlivoPay
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X GET 'https://api.klivopay.com.br/api/public/v1/bank-accounts?api_token=seu_token_aqui'
```

```javascript JavaScript
const response = await fetch(
  'https://api.klivopay.com.br/api/public/v1/bank-accounts?api_token=seu_token_aqui'
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

url = 'https://api.klivopay.com.br/api/public/v1/bank-accounts'
params = {'api_token': 'seu_token_aqui'}

response = requests.get(url, params=params)
print(response.json())
```


## Códigos de Resposta

<ResponseField name="200" type="OK">

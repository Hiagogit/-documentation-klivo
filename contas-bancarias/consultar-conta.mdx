## Visão Geral

Este endpoint retorna os detalhes completos de uma conta bancária ou chave PIX específica cadastrada.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/bank-accounts/{bankAccountHash}
```

## Parâmetros

<ParamField path="bankAccountHash" type="string" required>
  Hash identificador único da conta bancária
</ParamField>

<ParamField query="api_token" type="string" required>
  Seu token de autenticação da API KlivoPay
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X GET 'https://api.klivopay.com.br/api/public/v1/bank-accounts/bank123abc?api_token=seu_token_aqui'
```

```javascript JavaScript
const bankAccountHash = 'bank123abc';
const response = await fetch(
  `https://api.klivopay.com.br/api/public/v1/bank-accounts/${bankAccountHash}?api_token=seu_token_aqui`
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

bank_account_hash = 'bank123abc'
url = f'https://api.klivopay.com.br/api/public/v1/bank-accounts/{bank_account_hash}'
params = {'api_token': 'seu_token_aqui'}

response = requests.get(url, params=params)
print(response.json())
```

```php PHP
<?php
$bankAccountHash = 'bank123abc';
$url = "https://api.klivopay.com.br/api/public/v1/bank-accounts/{$bankAccountHash}?api_token=seu_token_aqui";
$response = file_get_contents($url);
print_r(json_decode($response));
?>
```

## Códigos de Resposta

<ResponseField name="200" type="OK">
  Conta bancária encontrada e retornada com sucesso
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Token de API inválido ou ausente
</ResponseField>

  Conta bancária não encontrada
</ResponseField>

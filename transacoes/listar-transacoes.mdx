## Visão Geral

Este endpoint permite listar todas as transações da sua conta, com opções de filtro por status e paginação para facilitar a navegação e geração de relatórios.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/transactions
```

## Parâmetros

<ParamField query="api_token" type="string" required>
  Seu token de autenticação da API KlivoPay
</ParamField>

<ParamField query="page" type="integer" default="1">
  Número da página para paginação
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Quantidade de itens por página (máximo: 100)
</ParamField>

<ParamField query="status" type="string">
  Filtrar por status da transação:
  - `pending` - Aguardando pagamento
  - `paid` - Pago
  - `canceled` - Cancelado
  - `refunded` - Reembolsado
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X GET 'https://api.klivopay.com.br/api/public/v1/transactions?api_token=seu_token_aqui&page=1&per_page=20&status=paid'
```

```javascript JavaScript
const params = new URLSearchParams({
  api_token: 'seu_token_aqui',
  page: 1,
  per_page: 20,
  status: 'paid'
});

const response = await fetch(`https://api.klivopay.com.br/api/public/v1/transactions?${params}`);
const data = await response.json();
console.log(data);
```

```python Python
import requests

url = 'https://api.klivopay.com.br/api/public/v1/transactions'
params = {
    'api_token': 'seu_token_aqui',
    'page': 1,
    'per_page': 20,
    'status': 'paid'
}

response = requests.get(url, params=params)
print(response.json())
```

```php PHP
<?php
$params = http_build_query([
    'api_token' => 'seu_token_aqui',
    'page' => 1,
    'per_page' => 20,
    'status' => 'paid'
]);

$url = 'https://api.klivopay.com.br/api/public/v1/transactions?' . $params;
$response = file_get_contents($url);
$data = json_decode($response);
print_r($data);
?>
```

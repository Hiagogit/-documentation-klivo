## Visão Geral

Este endpoint retorna todos os detalhes de uma transação específica, incluindo informações do cliente, dados de pagamento, status e histórico.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/transactions/{hash}
```

## Parâmetros

<ParamField path="hash" type="string" required>
  Hash identificador único da transação
</ParamField>

<ParamField query="api_token" type="string" required>
  Seu token de autenticação da API KlivoPay
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X GET 'https://api.klivopay.com.br/api/public/v1/transactions/abc123def456?api_token=seu_token_aqui'
```

```javascript JavaScript
const hash = 'abc123def456';
const apiToken = 'seu_token_aqui';

const response = await fetch(
  `https://api.klivopay.com.br/api/public/v1/transactions/${hash}?api_token=${apiToken}`
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

hash_id = 'abc123def456'
url = f'https://api.klivopay.com.br/api/public/v1/transactions/{hash_id}'
params = {'api_token': 'seu_token_aqui'}

response = requests.get(url, params=params)
print(response.json())
```

```php PHP
<?php
$hash = 'abc123def456';
$apiToken = 'seu_token_aqui';
$url = "https://api.klivopay.com.br/api/public/v1/transactions/{$hash}?api_token={$apiToken}";

$response = file_get_contents($url);
$data = json_decode($response);
print_r($data);
?>
```

</CodeGroup>

## Códigos de Resposta

<ResponseField name="200" type="OK">
  Transação encontrada e retornada com sucesso
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Token de API inválido ou ausente
</ResponseField>

<ResponseField name="404" type="Not Found">
  Transação não encontrada
</ResponseField>

<ResponseField name="500" type="Internal Server Error">
  Erro interno do servidor
</ResponseField>

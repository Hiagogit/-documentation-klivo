## Welcome to Klivopay API

Klivopay is a modern payment platform that allows you to process credit card and PIX transactions, manage products, offers, and more.

### Base URL

All API requests should be made to:

```
https://api.klivopay.com.br/api
```

## Getting Started

<Steps>
  <Step title="Get your API Token">
    Sign up at [Klivopay](https://klivopay.com.br) and get your API token from the dashboard
  </Step>
  <Step title="Make your first request">
    Check out our [Quickstart Guide](/quickstart) to make your first API call
  </Step>
  <Step title="Integrate webhooks">
    Set up [webhooks](/webhooks) to receive transaction updates
  </Step>
</Steps>

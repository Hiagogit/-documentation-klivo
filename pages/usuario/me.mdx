# Dados da Conta

Retorna as informações da conta associada ao token de API.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/me?api_token=SEU_TOKEN
```

## Exemplos de Requisição

<CodeGroup>
```bash cURL
curl -X GET "https://api.klivopay.com.br/api/public/v1/me?api_token=SEU_TOKEN"
```

```javascript JavaScript
const getAccount = async () => {
  const response = await fetch(
    'https://api.klivopay.com.br/api/public/v1/me?api_token=SEU_TOKEN'
  );
  return await response.json();
};
```
</CodeGroup>

## Resposta

```json
{
  "success": true,
  "data": {
    "id": 12345,
    "name": "Sua Empresa LTDA",
    "email": "contato@empresa.com",
    "document": "12345678000190",
    "account_status": "active",
    "created_at": "2025-01-01T00:00:00Z"
  }
}
```

## Casos de Uso

### Verificar Token

```javascript
async function isTokenValid() {
  try {
    const account = await getAccount();
    return account.success;
  } catch {
    return false;
  }
}
```

### Exibir Nome da Conta

```javascript
const account = await getAccount();
console.log(`Conta: ${account.data.name}`);
```

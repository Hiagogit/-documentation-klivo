# Ofertas

Ofertas permitem criar variações de preço para seus produtos, ideal para promoções, descontos e preços diferenciados.

## O que são Ofertas?

Uma oferta é uma variação de preço de um produto existente. Ela herda todas as características do produto pai (tipo, categoria, entrega) mas possui:

- Hash único
- Título próprio
- Preço diferenciado
- Imagem de capa própria (opcional)

## Estrutura de uma Oferta

```json
{
  "hash": "7becb",
  "product_hash": "7tjdfkshdv",
  "title": "Oferta de Lançamento - 40% OFF",
  "cover": "https://...",
  "amount": 29700,
  "created_at": "2026-03-16T10:00:00Z"
}
```

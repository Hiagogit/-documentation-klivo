# Atualizar Oferta

Atualiza as informações de uma oferta já criada.

## Endpoint

```
PUT https://api.klivopay.com.br/api/public/v1/products/:hash/offers?api_token=SEU_TOKEN
```

## Parâmetros do Body

<ParamField body="title" type="string">
  Novo título da oferta
</ParamField>

<ParamField body="amount" type="integer">
  Novo valor em centavos
</ParamField>

<ParamField body="cover" type="string">
  Nova URL da imagem
</ParamField>

## Exemplos de Requisição

<CodeGroup>
```bash cURL
curl -X PUT "https://api.klivopay.com.br/api/public/v1/products/7becb/offers?api_token=SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Super Promoção - 50% OFF",
    "amount": 24850
  }'
```

```javascript JavaScript
const updateOffer = async (offerHash, updates) => {
  const response = await fetch(
    `https://api.klivopay.com.br/api/public/v1/products/${offerHash}/offers?api_token=SEU_TOKEN`,
    {
      method: 'PUT',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(updates)
    }
  );
  return await response.json();
};
```
</CodeGroup>

## Resposta

```json
{
  "success": true,
  "data": {
    "hash": "7becb",
    "title": "Super Promoção - 50% OFF",
    "amount": 24850,
    "updated_at": "2026-03-16T15:00:00Z"
  }
}
```

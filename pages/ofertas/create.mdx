# Criar Oferta

Cria uma oferta promocional ou variação de preço para um produto existente.

## Endpoint

```
POST https://api.klivopay.com.br/api/public/v1/products/:hash/offers?api_token=SEU_TOKEN
```

## Parâmetros do Body

<ParamField body="title" type="string" required>
  Título da oferta
</ParamField>

<ParamField body="amount" type="integer" required>
  Valor da oferta em centavos
</ParamField>

<ParamField body="cover" type="string">
  URL da imagem de capa da oferta
</ParamField>

## Exemplos de Requisição

<CodeGroup>
```bash cURL
curl -X POST "https://api.klivopay.com.br/api/public/v1/products/7tjdfkshdv/offers?api_token=SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Oferta de Lançamento - 40% OFF",
    "amount": 29700,
    "cover": "https://exemplo.com/oferta.jpg"
  }'
```

```javascript JavaScript
const createOffer = async (productHash, offerData) => {
  const response = await fetch(
    `https://api.klivopay.com.br/api/public/v1/products/${productHash}/offers?api_token=SEU_TOKEN`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(offerData)
    }
  );
  return await response.json();
};

// Uso
const offer = await createOffer('7tjdfkshdv', {
  title: 'Black Friday - 50% OFF',
  amount: 24850
});
```
</CodeGroup>

## Resposta

```json
{
  "success": true,
  "data": {
    "hash": "7becb",
    "product_hash": "7tjdfkshdv",
    "title": "Oferta de Lançamento - 40% OFF",
    "amount": 29700,
    "cover": "https://exemplo.com/oferta.jpg",
    "created_at": "2026-03-16T10:00:00Z"
  }
}
```

## Usando a Oferta em Transações

Ao criar transação, use o `offer_hash`:

```json
{
  "amount": 29700,
  "offer_hash": "7becb",
  "payment_method": "credit_card",
  "customer": {...},
  "cart": [...]
}
```

# Listar Transações

Retorna uma lista de todas as transações processadas através da sua conta Klivopay.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/transactions?api_token=SEU_TOKEN
```

## Parâmetros de Query

<ParamField query="api_token" type="string" required>
  Token de autenticação da API
</ParamField>

<ParamField query="page" type="integer" default="1">
  Número da página para paginação
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Itens por página (máximo 100)
</ParamField>

<ParamField query="status" type="string">
  Filtrar por status: `pending`, `paid`, `failed`, `refunded`, `canceled`, `expired`
</ParamField>

<ParamField query="payment_method" type="string">
  Filtrar por método: `pix` ou `credit_card`
</ParamField>

<ParamField query="start_date" type="string">
  Data inicial (YYYY-MM-DD)
</ParamField>

<ParamField query="end_date" type="string">
  Data final (YYYY-MM-DD)
</ParamField>

## Exemplos de Requisição

<CodeGroup>
```bash Listar Todas
curl -X GET "https://api.klivopay.com.br/api/public/v1/transactions?api_token=SEU_TOKEN"
```

```bash Com Filtros
curl -X GET "https://api.klivopay.com.br/api/public/v1/transactions?api_token=SEU_TOKEN&status=paid&payment_method=pix&per_page=50"
```

```bash Por Período
curl -X GET "https://api.klivopay.com.br/api/public/v1/transactions?api_token=SEU_TOKEN&start_date=2026-03-01&end_date=2026-03-31"
```

```javascript JavaScript
const listTransactions = async (filters = {}) => {
  const params = new URLSearchParams({
    api_token: 'SEU_TOKEN',
    ...filters
  });

  const response = await fetch(
    `https://api.klivopay.com.br/api/public/v1/transactions?${params}`
  );

  return await response.json();
};

// Uso
const transactions = await listTransactions({
  status: 'paid',
  per_page: 50
});
```


## Filtros e Ordenação

### Filtrar por Status

```bash
GET /transactions?api_token=TOKEN&status=paid
```

Status disponíveis:
- `pending` - Aguardando pagamento
- `paid` - Pago
- `failed` - Falhou
- `refunded` - Reembolsado
- `canceled` - Cancelado
- `expired` - Expirado (PIX)

### Filtrar por Método de Pagamento

```bash
GET /transactions?api_token=TOKEN&payment_method=pix
```

### Filtrar por Período

```bash
GET /transactions?api_token=TOKEN&start_date=2026-03-01&end_date=2026-03-31
```

### Combinar Filtros

```bash
GET /transactions?api_token=TOKEN&status=paid&payment_method=credit_card&start_date=2026-03-01
```

## Paginação

### Navegando entre Páginas

<CodeGroup>
```bash Página 1
curl "https://api.klivopay.com.br/api/public/v1/transactions?api_token=TOKEN&page=1&per_page=20"
```

```bash Página 2
curl "https://api.klivopay.com.br/api/public/v1/transactions?api_token=TOKEN&page=2&per_page=20"
```

```javascript JavaScript
async function getAllTransactions() {
  const allTransactions = [];
  let page = 1;
  let hasMore = true;

  while (hasMore) {
    const response = await fetch(
      `https://api.klivopay.com.br/api/public/v1/transactions?api_token=TOKEN&page=${page}&per_page=100`
    );

    const { data, pagination } = await response.json();

    allTransactions.push(...data);

    hasMore = page < pagination.total_pages;
    page++;
  }

  return allTransactions;
}
```
</CodeGroup>

## Casos de Uso

### Listar Transações Pagas Hoje

```javascript
const today = new Date().toISOString().split('T')[0];

const response = await fetch(
  `https://api.klivopay.com.br/api/public/v1/transactions?api_token=TOKEN&status=paid&start_date=${today}&end_date=${today}`
);

const { data } = await response.json();
console.log(`${data.length} transações pagas hoje`);
```

### Gerar Relatório Mensal

```python
from datetime import datetime, timedelta

# Primeiro dia do mês
start_date = datetime.now().replace(day=1).strftime('%Y-%m-%d')

# Último dia do mês
end_date = datetime.now().strftime('%Y-%m-%d')

url = f"https://api.klivopay.com.br/api/public/v1/transactions"
params = {
    "api_token": "TOKEN",
    "status": "paid",
    "start_date": start_date,
    "end_date": end_date,
    "per_page": 100
}

response = requests.get(url, params=params)
transactions = response.json()['data']

total = sum(t['amount'] for t in transactions)
print(f"Total: R$ {total/100:.2f}")
```

## Observações

<Info>
  Por padrão, as transações são ordenadas por data de criação (mais recentes primeiro).
</Info>

<Note>
  O limite máximo de itens por página é 100. Para consultas maiores, use paginação.
</Note>

<Tip>
  Para obter detalhes completos de uma transação específica, use o endpoint [GET /transactions/:hash](/pages/transacoes/get).
</Tip>

# Categorias de Produtos

Retorna a lista completa de categorias disponíveis para classificar produtos.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/products/categories?api_token=SEU_TOKEN
```

## Exemplos de Requisição

<CodeGroup>
```bash cURL
curl -X GET "https://api.klivopay.com.br/api/public/v1/products/categories?api_token=SEU_TOKEN"
```

```javascript JavaScript
const getCategories = async () => {
  const response = await fetch(
    'https://api.klivopay.com.br/api/public/v1/products/categories?api_token=SEU_TOKEN'
  );
  return await response.json();
};
```
</CodeGroup>

## Usando Categorias

Use o `id` da categoria ao criar produtos:

```json
{
  "title": "Curso de React",
  "id_category": 1,
  "product_type": "digital",
  "amount": 49700
}
```

# Consultar Produto

Retorna os detalhes completos de um produto específico usando seu hash único.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/products/:hash?api_token=SEU_TOKEN
```

## Exemplos de Requisição

<CodeGroup>
```bash cURL
curl -X GET "https://api.klivopay.com.br/api/public/v1/products/7tjdfkshdv?api_token=SEU_TOKEN"
```

```javascript JavaScript
const getProduct = async (hash) => {
  const response = await fetch(
    `https://api.klivopay.com.br/api/public/v1/products/${hash}?api_token=SEU_TOKEN`
  );
  return await response.json();
};
```
</CodeGroup>

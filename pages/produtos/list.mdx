# Listar Produtos

Retorna uma lista de todos os produtos cadastrados na sua conta Klivopay.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/products?api_token=SEU_TOKEN
```

## Exemplos de Requisição

<CodeGroup>
```bash cURL
curl -X GET "https://api.klivopay.com.br/api/public/v1/products?api_token=SEU_TOKEN"
```

```javascript JavaScript
const listProducts = async () => {
  const response = await fetch(
    'https://api.klivopay.com.br/api/public/v1/products?api_token=SEU_TOKEN'
  );
  return await response.json();
};
```

```python Python
import requests

url = "https://api.klivopay.com.br/api/public/v1/products"
params = {"api_token": "SEU_TOKEN"}

response = requests.get(url, params=params)
products = response.json()
```
</CodeGroup>

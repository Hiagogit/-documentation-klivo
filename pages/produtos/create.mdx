# Criar Produto

Cadastra um novo produto digital ou físico na sua conta Klivopay.

## Endpoint

```
POST https://api.klivopay.com.br/api/public/v1/products?api_token=SEU_TOKEN
```

## Parâmetros do Body

<ParamField body="title" type="string" required>
  Título do produto (máximo 255 caracteres)
</ParamField>

<ParamField body="product_type" type="string" required>
  Tipo do produto: `"digital"` ou `"fisico"`
</ParamField>

<ParamField body="delivery_type" type="integer" required>
  Tipo de entrega:
  - `1` = Área de membros da plataforma
  - `2` = Área externa / URL customizada
</ParamField>

<ParamField body="id_category" type="integer" required>
  ID da categoria (consulte via `/products/categories`)
</ParamField>

<ParamField body="amount" type="integer" required>
  Valor do produto em centavos
</ParamField>

<ParamField body="cover" type="string">
  URL da imagem de capa do produto
</ParamField>

<ParamField body="sale_page" type="string">
  URL da página de vendas
</ParamField>

<ParamField body="payment_type" type="integer" default="1">
  Tipo de pagamento:
  - `1` = Pagamento único
  - `2` = Recorrente (assinatura)
</ParamField>

## Exemplos de Requisição

<CodeGroup>
```bash Produto Digital
curl -X POST "https://api.klivopay.com.br/api/public/v1/products?api_token=SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Curso de Desenvolvimento Web",
    "product_type": "digital",
    "delivery_type": 1,
    "id_category": 1,
    "amount": 49700,
    "cover": "https://exemplo.com/imagens/curso.jpg",
    "sale_page": "https://seusite.com/curso"
  }'
```

```bash Produto Físico
curl -X POST "https://api.klivopay.com.br/api/public/v1/products?api_token=SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Livro Impresso - JavaScript",
    "product_type": "fisico",
    "delivery_type": 2,
    "id_category": 6,
    "amount": 8900
  }'
```

```javascript JavaScript
const createProduct = async (productData) => {
  const response = await fetch(
    'https://api.klivopay.com.br/api/public/v1/products?api_token=SEU_TOKEN',
    {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(productData)
    }
  );

  return await response.json();
};

// Uso
const product = await createProduct({
  title: 'Curso de React Avançado',
  product_type: 'digital',
  delivery_type: 1,
  id_category: 1,
  amount: 49700
});
```

```python Python
import requests

url = "https://api.klivopay.com.br/api/public/v1/products"
params = {"api_token": "SEU_TOKEN"}

payload = {
    "title": "Curso de Python",
    "product_type": "digital",
    "delivery_type": 1,
    "id_category": 1,
    "amount": 39700,
    "sale_page": "https://seusite.com/python"
}

response = requests.post(url, params=params, json=payload)
data = response.json()
```
</CodeGroup>

## Códigos de Status HTTP

| Código | Descrição |
|--------|-----------|
| 201 | Produto criado com sucesso |
| 400 | Dados inválidos |
| 401 | Token inválido ou ausente |
| 422 | Erro de validação |
| 500 | Erro interno do servidor |

## Erros Comuns

<AccordionGroup>
  <Accordion title="400 - Categoria Inválida">
    ```json
    {
      "success": false,
      "error": "Invalid category",
      "details": {
        "id_category": "Category not found"
      }
    }
    ```

    **Solução**: Consulte as categorias disponíveis via `/products/categories`.
  </Accordion>

  <Accordion title="422 - Valor Inválido">
    ```json
    {
      "success": false,
      "error": "Validation failed",
      "details": {
        "amount": "Amount must be greater than 100"
      }
    }
    ```

    **Solução**: O valor mínimo é R$ 1,00 (100 centavos).
  </Accordion>
</AccordionGroup>

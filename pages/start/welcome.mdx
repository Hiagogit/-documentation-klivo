## Recursos Principais

### Métodos de Pagamento

- **PIX**: Pagamentos instantâneos via PIX com QR Code
- **Cartão de Crédito**: Parcelamento em até 12x com ou sem juros

### Funcionalidades

- Gestão completa de produtos e ofertas
- Cálculo automático de parcelamento
- Webhooks para notificações em tempo real
- Reembolsos totais e parciais
- API REST completa e documentada

### Segurança

- Autenticação via token API
- Comunicação via HTTPS
- Conformidade com padrões PCI

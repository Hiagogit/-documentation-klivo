# Checkout

O sistema de checkout permite criar links de pagamento personalizados para seus produtos.

## Como Funciona

1. Crie um checkout (via dashboard ou API)
2. Receba um hash único
3. Compartilhe o link com seu cliente
4. Cliente realiza o pagamento
5. Receba webhook de confirmação

## Endpoint Disponível

<Card title="Inicializar Checkout" icon="shopping-cart" href="/pages/checkout/initialize">
  `GET /checkout/:hash`

  Carrega dados do checkout para exibir ao cliente
</Card>

## Link de Checkout

```
https://checkout.klivopay.com.br/:hash
```

<Note>
  Esta rota **não requer autenticação** pois é acessada diretamente pelo cliente.
</Note>

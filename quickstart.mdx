## Quick Integration Guide

This guide will help you process your first payment with Klivopay in just a few steps.

<Steps>
  <Step title="Get Your API Token">
    Follow the [Authentication](/authentication) guide to get your API token.
  </Step>

  <Step title="Create a Product">
    First, create a product in your Klivopay account:

    ```bash
    curl -X POST "https://api.klivopay.com.br/api/public/v1/products?api_token=YOUR_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "title": "My First Product",
        "product_type": "digital",
        "delivery_type": 1,
        "id_category": 1,
        "amount": 10000
      }'
    ```

    <Info>Amounts are always in cents. 10000 = R$ 100.00</Info>
  </Step>

  <Step title="Create an Offer">
    Create an offer for your product using the product hash from the previous response:

    ```bash
    curl -X POST "https://api.klivopay.com.br/api/public/v1/products/PRODUCT_HASH/offers?api_token=YOUR_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "title": "Launch Offer",
        "amount": 7900
      }'
    ```
  </Step>

  <Step title="Process a Payment">
    Now process a payment using the offer hash:

    ```bash
    curl -X POST "https://api.klivopay.com.br/api/public/v1/transactions?api_token=YOUR_TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "amount": 7900,
        "offer_hash": "OFFER_HASH",
        "payment_method": "credit_card",
        "customer": {
          "name": "João Silva",
          "email": "joao@example.com",
          "phone_number": "11999999999",
          "document": "12345678900",
          "street_name": "Rua Exemplo",
          "number": "123",
          "neighborhood": "Centro",
          "city": "São Paulo",
          "state": "SP",
          "zip_code": "01234567"
        },
        "card": {
          "number": "4111111111111111",
          "holder_name": "JOAO SILVA",
          "exp_month": 12,
          "exp_year": 2025,
          "cvv": "123"
        },
        "cart": [{
          "product_hash": "PRODUCT_HASH",
          "title": "My First Product",
          "price": 7900,
          "quantity": 1,
          "tangible": false
        }],
        "installments": 1,
        "transaction_origin": "api"
      }'
    ```
  </Step>
</Steps>

## PIX Payments

For PIX payments, simply change the `payment_method` to `"pix"` and remove the `card` object:

```json
{
  "payment_method": "pix",
  "expire_in_days": 1,
  // ... other fields
}
```

The API will return a PIX QR code and payment instructions.

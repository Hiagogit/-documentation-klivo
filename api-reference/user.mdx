## Get Account Info

Returns information about the authenticated account.

```bash
GET /public/v1/me?api_token=YOUR_TOKEN
```

### Response Example

```json
{
  "id": 123,
  "name": "Company Name",
  "email": "contact@company.com",
  "api_token": "your_token_here",
  "created_at": "2024-01-01T00:00:00Z"
}
```

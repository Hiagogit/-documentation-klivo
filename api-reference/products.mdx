## List Products

Get all products from your account.

```bash
GET /public/v1/products?api_token=YOUR_TOKEN
```

## Get Product

Get a specific product by hash.

```bash
GET /public/v1/products/:hash?api_token=YOUR_TOKEN
```

## Create Product

<RequestExample>

```bash cURL
curl -X POST "https://api.klivopay.com.br/api/public/v1/products?api_token=YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Product Title",
    "product_type": "digital",
    "delivery_type": 1,
    "id_category": 1,
    "amount": 10000
  }'
```

</RequestExample>

### Request Body

<ParamField body="title" type="string" required>
  Product title
</ParamField>

<ParamField body="product_type" type="string" required>
  Product type: `digital` or `fisico`
</ParamField>

<ParamField body="delivery_type" type="integer" required>
  - `1`: Platform member area
  - `2`: External area
</ParamField>

<ParamField body="id_category" type="integer" required>
  Category ID (see GET /products/categories)
</ParamField>

<ParamField body="amount" type="integer" required>
  Price in cents
</ParamField>

<ParamField body="cover" type="string">
  Cover image URL
</ParamField>

<ParamField body="sale_page" type="string">
  Sales page URL
</ParamField>

## Update Product

Update an existing product.

```bash
PUT /public/v1/products/:hash?api_token=YOUR_TOKEN
```

## Get Categories

List all available product categories.

```bash
GET /public/v1/products/categories?api_token=YOUR_TOKEN
```

## Create Offer

Create a new offer for a product.

<RequestExample>

```bash cURL
curl -X POST "https://api.klivopay.com.br/api/public/v1/products/PRODUCT_HASH/offers?api_token=YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Launch Offer",
    "amount": 5000
  }'
```

</RequestExample>

### Request Body

<ParamField body="title" type="string" required>
  Offer title
</ParamField>

<ParamField body="amount" type="integer" required>
  Offer price in cents
</ParamField>

<ParamField body="cover" type="string">
  Cover image URL
</ParamField>

## Update Offer

Update an existing offer.

```bash
PUT /public/v1/products/:hash/offers?api_token=YOUR_TOKEN
```

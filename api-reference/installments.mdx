## Calculate Installments

Get available installment plans for a given amount.

```bash
GET /public/v1/installments?api_token=YOUR_TOKEN&amount=10000&interest_type=simple
```

### Query Parameters

<ParamField query="api_token" type="string" required>
  Your API authentication token
</ParamField>

<ParamField query="amount" type="integer" required>
  Amount in cents
</ParamField>

<ParamField query="interest_type" type="string" required>
  Interest calculation type: `simple` or `compound`
</ParamField>

### Response Example

```json
{
  "installments": [
    {
      "number": 1,
      "amount": 10000,
      "total": 10000,
      "interest_rate": 0
    },
    {
      "number": 2,
      "amount": 5100,
      "total": 10200,
      "interest_rate": 2
    }
    // ... more installments
  ]
}
```

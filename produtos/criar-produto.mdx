## Visão Geral

Este endpoint permite cadastrar um novo produto na plataforma KlivoPay. Produtos são itens que você vende e podem ser digitais ou físicos.

## Endpoint

```
POST https://api.klivopay.com.br/api/public/v1/products
```

## Parâmetros

<ParamField body="api_token" type="string" required>
  Seu token de autenticação da API KlivoPay
</ParamField>

<ParamField body="title" type="string" required>
  Título/nome do produto
</ParamField>

<ParamField body="cover" type="string" required>
  URL da imagem de capa do produto
</ParamField>

<ParamField body="amount" type="integer" required>
  Valor do produto em centavos. Exemplo: R$ 299,00 = `29900`
</ParamField>

<ParamField body="payment_type" type="integer" required>
  Tipo de pagamento: `1` = Pagamento único
</ParamField>

<ParamField body="product_type" type="string" required>
  Tipo do produto: `digital` ou `fisico`
</ParamField>

<ParamField body="id_category" type="integer" required>
  ID da categoria do produto (use o endpoint de categorias para consultar)
</ParamField>

<ParamField body="sale_page" type="string">
  URL da página de vendas do produto
</ParamField>

<ParamField body="delivery_type" type="string">
  Tipo de entrega para produtos digitais: `email`, `download`, etc.
</ParamField>

<CodeGroup>

```bash cURL
  -H 'Content-Type: application/json' \
  -d '{

    "title": "Curso de Marketing Digital",

    "payment_type": 1,
    "product_type": "digital",
    "delivery_type": "email",
    "id_category": 5,
    "amount": 29900
  }'
```

```python Python
import requests

url = 'https://api.klivopay.com.br/api/public/v1/products'
payload = {
    'api_token': 'seu_token_aqui',
    'title': 'Curso de Marketing Digital',
    'cover': 'https://exemplo.com/capa.jpg',
    'payment_type': 1,
    'product_type': 'digital',
    'id_category': 5,
    'amount': 29900
}

response = requests.post(url, json=payload)
print(response.json())
```

```php PHP
<?php
$url = 'https://api.klivopay.com.br/api/public/v1/products';
$data = [
    'api_token' => 'seu_token_aqui',
    'title' => 'Curso de Marketing Digital',
    'cover' => 'https://exemplo.com/capa.jpg',
    'payment_type' => 1,
    'product_type' => 'digital',
    'id_category' => 5,
    'amount' => 29900
];

$options = [
    'http' => [
        'header'  => "Content-Type: application/json\r\n",
        'method'  => 'POST',
        'content' => json_encode($data)
    ]
];

$context = stream_context_create($options);
$result = file_get_contents($url, false, $context);
print_r(json_decode($result));
?>
```

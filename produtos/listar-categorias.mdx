## Visão Geral

Este endpoint retorna a lista de todas as categorias disponíveis para classificação de produtos.

## Endpoint

```
GET https://api.klivopay.com.br/api/public/v1/products/categories
```

## Parâmetros

<ParamField query="api_token" type="string" required>
  Seu token de autenticação da API KlivoPay
</ParamField>

## Exemplo de Requisição

<CodeGroup>

```bash cURL
curl -X GET 'https://api.klivopay.com.br/api/public/v1/products/categories?api_token=seu_token_aqui'
```

```python Python
import requests
```
